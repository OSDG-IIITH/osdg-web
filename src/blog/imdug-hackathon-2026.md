We at OSDG recently collaborated with the [Indic MediaWiki Developers User Group (IMDUG)](https://meta.wikimedia.org/wiki/Indic_MediaWiki_Developers_User_Group) and [Open Knowledge Initiatives (OKI), IIIT Hyderabad](https://meta.wikimedia.org/wiki/IIITH-OKI) to help conduct their latest hackathon, the **Indic Wikimedia Hackathon Hyderabad 2026**, held from 26th to 28th June 2026.

### Structure

Unlike regular "competitive" hackathons, Wikimedia hackathons are a place where people work together to build tools for the MediaWiki ecosystem. This seems like a pretty fun idea, and we at OSDG look forward to experiment with this format in the coming year.

For more details, you can check out the [Meta-Wiki Event Page](https://meta.wikimedia.org/wiki/Main_Page) and the [Etherpad](https://etherpad.wikimedia.org/).

## Project Showcase

### 1. New Wikipedia Editor Onboarding Platform

A centralized platform designed to streamline the onboarding experience for new Wikimedia volunteers while reducing repetitive administrative overhead for community organizers.

**Links:** [GitHub Repository](https://github.com/Agamya-Samuel/twa2) | [Phabricator T430437](https://phabricator.wikimedia.org/T430437) | [Phabricator T430438](https://phabricator.wikimedia.org/T430438)

### 2. WikiLinkua

A web-based language learning platform focused on building the existing vocabulary of the learner using Wikidata Lexemes and Wiktionary.

**Links:** [GitHub Repository](https://github.com/rohan2023101003/wikilinkua/) | [Phabricator T425045](https://phabricator.wikimedia.org/T425045)

### 3. LinguaLibre

A collaborative, multilingual, audiovisual corpus project by Wikimédia France aimed at expanding knowledge about languages and supporting online language communities.

**Links:** [GitLab Repository](https://gitlab.wikimedia.org/repos/wikimedia-france/lingua-libre/lingua-libre) | [Phabricator Project Page](https://phabricator.wikimedia.org/tag/lingua-libre/)

### 4. OpenSpeaks

A suite of tools for community-based language and cultural documentation. It includes *Subtitler* (subtitle editing), *Tome* (metadata management and video uploading), and *Bento* (file organization and analysis).

**Links:** [Subtitler Phabricator](https://phabricator.wikimedia.org/project/profile/8525/) | [Tome Phabricator](https://phabricator.wikimedia.org/project/profile/8714/)

### 5. Translate Tagger

An automation tool that applies internationalization markup on pages to be made translatable on multilingual wikis like Wikimedia Commons, reducing hours of manual formatting down to seconds.

**Links:** [GitHub Repository](https://github.com/indictechcom/TranslateTagger) | [Phabricator Project Page](https://phabricator.wikimedia.org/tag/tool-translatetagger/)

### 6. Montage

A photo evaluation tool for the Wiki Loves Monuments competition. The team added an aggregate scoring final round, dark mode functionality, image zooming, parameterized round validation, and expanded round data views.

**Links:** [GitHub Repository](https://github.com/hatnote/montage/)

### 7. Scribe

An offline virtual keyboard that integrates Wikidata. The team implemented clipboard functionality and resolved deployment bugs.

### 8. Deployr

A deployment tool built to automate the workflow and database connections for deploying applications to Wikimedia Toolforge.

### 9. Clip2Commons

A browser-based video upload tool for Wikimedia Commons that allows users to sign in and upload files directly without server queues.

**Links:** [GitLab Repository](https://gitlab.wikimedia.org/toolforge-repos/clip2commons)

### 10. Wikievol Tool

A development utility targeting Wiki evolution analytics and platform tracking metrics.

**Links:** [GitHub Repository](https://github.com/mahi01agarwal/Wikievol)

### 11. NPOV Drift Detector

A research tool that monitors a Wikipedia article’s full edit history to detect directional and persistent neutral point-of-view (NPOV) drift windows for human review.

**Links:** [GitHub Repository](https://github.com/mishradev1/npov-drift-detector) | [Phabricator T428896](https://phabricator.wikimedia.org/T428896)

### 12. Observability Tool

An infrastructure tool designed to improve tracking, monitoring, and scope visibility within Toolforge environments.

**Links:** [GitLab Repository](https://gitlab.wikimedia.org/punit/toolscope)

### 13. Wikisource Reader App & Metadata API

An Android mobile application and metadata indexer that allows users to search, download, and read public domain works hosted on the Wikisource project.

**Links:** [GitHub Application Repository](https://github.com/cis-india/Wikisource-Reader) | [GitLab Indexer Repository](https://gitlab.wikimedia.org/toolforge-repos/wsindex)

### 14. Wikimedia Commons App

An Android application enabling users to upload freely licensed media, locate nearby points of interest requiring photographs, and track their contributions.

**Links:** [GitHub Repository](https://github.com/commons-app/apps-android-commons.git)

### 15. LanguageSelector Rewrite

A reconstruction of the Universal Language Selector (ULS) settings interface using Vue.js and the Codex design system to provide responsive mobile support for Wikimedia projects.

**Links:** [Phabricator T430255](https://phabricator.wikimedia.org/T430255)

### 16. WikiSage / WikiGPT

An artificial intelligence integration project focused on parsing, summarizing, or querying wiki knowledge bases.
